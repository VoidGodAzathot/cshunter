### Api Device Id

### Получить идентификатор
- **Функция:** `get_device_id`
- **Пример использования:**
```ts
const device_id: string = await invoke("get_device_id");
```

### Получить Ip адрес
- **Функция:** `get_ip_addr`
- **Пример использования:**
```ts
const device_id: string | undefined = await invoke("get_ip_addr");
```
